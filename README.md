# agePlayr: The age defining music app

Simply input the year you were born and log into Spotify, it handles the rest!

## Requirements
 * Flask 
 
  `pip install flask --user
 * Spotipy 
 
  `pip install spotipy`
 * Requests
 
  `pip install requests`
  
## To Run:
  * clone repo
  * Run main.py in terminal
  
   `python main.py`
  * look at web browser
