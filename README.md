# agePlayr: The age defening music app

Simply input the year you were born and log into spotify, it handels the rest!

## Requierments
 * Flask 
 
  `pip install flask --user
 * Spotipy 
 
  `pip install spotipy`
 * Requests
 
  `pip install requests`
  
## To Run:
  * clone repo
  * Run main.py in terminal
  
   `python main.py`
  * look at web browser
